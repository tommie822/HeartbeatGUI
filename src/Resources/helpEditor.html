In here you can find some documentation for the following functions of the program.
  • Import
  • Connect
  • Editor

Import

With the import function you can read the data from sd-card of the standalone device. You can also open all the .heartrate files that where made by saving the real-time data.

To use this function of the program you will need to do the following steps:

  1. Click on "Import"
  2. Find the .heartrate file you want to open. This can be on your computer or on the sd-card
  3. Click on "Open"
  4. If everything was done right you should now have all the data from the file
If you want to link names to all the wristband-IDs you can click on Editor.

Connect

With this function you can connect your standalone device to the computer so that you can see real-time data from the patients and get alerts or other usefull information.

To connect the standalone device you will need to connect it with the delivered USB-Cable. When connected you will need to do the following steps:

  1. Click on "Connect"

Editor

This function is to link names to every wristband-ID. So that you don't have to learn which person has which ID-number.

To link names to each of the wristbands you need to do the following steps:

  1. Click on "Editor"
  2. Select the wristband you want to link a name to
  3. Now click on the right textfield and type in the name you want to give to the ID
  4. Do this for all the ID's you want to give a name
  5. You can now close the editor by clicking on "close"
You now have succesfully linked each ID to a name. And if everything went correct you can see the list of patients being updated with their names.
