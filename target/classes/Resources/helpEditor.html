General

The program is designed to be user as users-friendly as possible. On the left side you can see in one overview all patients that are available to lookup. When a patient is clicked and you can see their heartbeat data in a nice linechart. If you want to select specific dates to look up the patients data more closely. You can use the Time span selector. If you want to change the names that are linked to the wristbands you can click on the Editor in here you can change all the names and setup when an alarm must go of for each of the patients. If you are done measuring all the patients you can save the data by following this guide Save. Ofcourse you want to load the saved data, this can be done by using the Import function. In here you can also import the data directly from the base station.

Connect

To connect the standalone device you will need to connect it with the delivered USB-Cable. After connecting follow the steps beneath.

  1. Click on "Connect"
  2. Select "Arduino Uno"
  3. Click on "Connect"
You are now connected to the standalone device and new patient should pop up soon. If you want to disable real-time data just toggle the button on the top of the linechart.

Import

To import new data in the program follow the following steps.

  1. Click on "Import"
  2. Find the .heart or .txt file you want to open
  3. Click on "Open"
You should now be able to see all the new data in the program.

Editor

To link names to each of the wristbands and select there warning trigger, follow these steps.

  1. Click on "Editor"
  2. Select the wristband-ID you want to link a name too
  3. Type the name of the patient in the appropriate textfield
  4. Type in the minimum and maximum heartbeat of the patient
  5. Close the editor
You now have succesfully linked each ID to a name. You should be able to see all the names in the patientlist. Also when the patients heartbeat goes below or above the given limits the program will give a warning.

Editor

If you want to save you need to do the following.

  1. Click on "Save"
  2. Select all patients u want to save
  3. Click on "Save" and select a file where you want to save to
You now have succesfully saved the patients to a file.

Time span selector

Select two dates where you want to know the values between. After selecting the dates click on reload linechart. If no dates are selected the program will assume that there are no dates.
