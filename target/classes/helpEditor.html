Connect

Clicking this button will give you a new screen. In here you can find a list of all devices. If you want to connect to your standalone device, select "Arduino Uno". When selected, you will need to click on the connect button and the window will automatically close.

To connect the standalone device you will need to connect it with the delivered USB-Cable. When connected you will need to do the following steps:

  1. Click on "Connect"
  2. Select "Arduino Uno"
  3. Click on "Connect" but this time in the new window
You are now connected to the standalone device and new patient should pop up soon in the patientlist. You should also be able to see real-time data of the patients. If you are not seeing real-time data but still sees all the graphs from the patients. Be sure to enable the real-time data by clicking on the real-time button.

Import

On clicking this button you will get a new window where you can select a heartrate file that you want to read out.

To use this function of the program you will need to do the following steps:

  1. Click on "Import"
  2. Find the .heartrate file you want to open.
  3. Click on "Open"
If everything was done right you should now have imported all the data from the file.

Editor

This function is to link names to every wristband-ID. So that you don't have to learn which person has which ID-number.

To link names to each of the wristbands you need to do the following steps:

  1. Click on "Editor"
  2. Select the wristband-ID you want to link a name to
  3. Now click on the right textfield and type in the name you want to give to that specific ID
  4. Do this for all the ID's you want to give a name
  5. You can now close the editor
You now have succesfully linked each ID to a name. You should now be able to see all the names in the patientlist.
